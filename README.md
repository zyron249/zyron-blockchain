# ZYRON Blockchain

## What is ZYRON?
ZYRON is a custom Layer-1 blockchain built with Cosmos SDK. Native token: ZYN (base denom: uzyn). Chain ID: zyron-testnet-1

## Quick Start

### Prerequisites
- Go 1.22+
- Ignite CLI

### Install Ignite
```bash
curl https://get.ignite.network/ignite! | bash
```

### Build the chain
Follow the instructions in docs/ to initialize.
